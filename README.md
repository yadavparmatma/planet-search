A Planet Search App
     
        Search by Planet Name
        
* How to build
        
        clone the repossitory useing
            git clone https://github.com/yadavparmatma/planet-search.git
         
         cd planet-search
             -> npm build


* How to run
        
        npm start
        
        then hit the browser http://localhost:8080
        

* How to Use
        
        Login using Star War character resource
